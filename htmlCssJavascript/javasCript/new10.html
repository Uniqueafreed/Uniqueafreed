Prompt dilog Box:-

It is useful dialog box for yout to pop-up a text bos to get user input
and also enables to interact with user. The user needs to fill in the field and then click Ok.
This dialog bos is displayed using a method called prompt()
which takes two paremeters.


The parameters are as follows:-

  1. A label which you want to display in the text box and--

  2. A default string to display in the text box.
This dialog box has two buttons, OK and Cancel.
If the user clicks the OK button, the window method prompt()
will return the entered value from the text box. If the
user clicks the Cancel button, The window method prompt() returns null.
